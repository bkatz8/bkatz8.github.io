Magic Moments

One of my first projects was working on the UX/UI for FullStory's "Magic Moments." What are magic moments you ask? They're moments of 'magical' insight that FullStory pulls from a user session to better help you understand how they're using you're website!

 

The sidebar circled on the right is a chronological list of all the events that have occured in a session. Rage Clicks show when a user is angry with something in your website! Dead Clicks show when a clicked link isn't working. JS Errors show errors in your Javascript that a user has encountered.
